# springboot-cli
springboot-cli for quickly build project base springboot

springboot-cli主要用于能够快速搭建基于springboot的项目框架，例如SpringBoot+SpringCloud，SpringBoot+SpringSecurity等，方便开发者们能够开箱即用。

# demo

- Oauth2
    - oauth2-authorization-server
    - oauth2-resource

- SSO
    - sso-auth-client01
    - sso-auth-client02
    - sso-auth-server

- SpringBoot + RocketMQ
    - rocketmq-producer
    - rocketmq-consumer
